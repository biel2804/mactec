# Assets obrigatórios de ícone — MacTec Messenger

Esta pasta deve conter **apenas ícones finais de boa qualidade** antes do build de produção.

## Arquivos esperados
1. `icon-192.png` — 192x192, PNG, fundo transparente, versão principal do app.
2. `icon-512.png` — 512x512, PNG, versão principal em alta resolução.
3. `icon-maskable-512.png` — 512x512, PNG, com área segura para recorte maskable Android.

## Observação
- Neste saneamento os placeholders binários foram removidos para evitar merge de assets temporários.
- Adicione os três arquivos acima antes do empacotamento final do APK.
